Design Summary

Signal Visual Treatment Meaning
Card green border 1.5px solid #3FB950 + subtle glow "This fixture has ≥1 value bet — look here first"
Card grey border 1px solid #30363D (default) "No value bets, but model's best guess is marked"
★ badge + green double ring ★ prefix + 2px #3FB950 + 4px glow ring + outer glow "Bet THIS — value bet, model's top pick"
★ badge + blue ring ★ prefix + 2px #58A6FF + blue glow "Model's best guess, but below your threshold"
Plain badge (no star, no ring) Background colour only (green/yellow/red/grey) "Other markets — colour shows edge direction"

Hover any ★ badge to see the enriched tooltip with edge %, model probability, and confidence level.
