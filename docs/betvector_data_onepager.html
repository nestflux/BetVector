The 8 Data Sources
Football-Data.co.uk
Six seasons of EPL match results, half-time scores, referee names, and opening odds from Bet365, Pinnacle, and William Hill.
6 SEASONS · CSV
Football-Data.org API
Real-time result updates. Solves the 2–4 day lag on the CSV source — results are available minutes after the final whistle.
REAL-TIME · JSON
Understat
Advanced analytics: Expected Goals (xG), non-penalty xG, pressing intensity (PPDA), deep completions, set-piece vs open-play xG breakdown.
ADVANCED STATS · DAILY
The Odds API
Live odds from 50+ bookmakers across UK, US, and European markets. Covers 1X2, Over/Under 1.5, 2.5, 3.5, and BTTS markets.
50+ BOOKMAKERS · LIVE
API-Football
Full fixture list with kickoff times, real-time scores, injury and suspension reports, and team crest images.
FIXTURES · INJURIES
ClubElo
Daily Elo ratings for every European club. Long-term team strength signal — especially valuable for newly promoted sides early in a season.
ELO RATINGS · DAILY
Transfermarkt
Squad market values in EUR for all EPL teams. Richer squads systematically outperform — value ratio vs opponent is a strong predictor.
SQUAD VALUES · WEEKLY
Open-Meteo
Hourly weather at every EPL stadium at kickoff. Temperature, wind, precipitation, and humidity — heavy rain and wind measurably reduce goal counts.
WEATHER · PER MATCH
Injury Flags
Owner-entered flags for key player absences. Each absence is scored 0–1 for impact; a flag ≥ 0.7 triggers a binary "key player out" signal in the model.
MANUAL · AS NEEDED
The Intelligence Layer
Raw data is transformed into 50+ features before the model ever sees it. Features fall into six categories, each adding a different dimension of signal.

The Prediction Model
A Poisson regression model predicts how many goals each team is likely to score — separately for home and away — based on ~25 of the features above. From those two numbers, every possible scoreline (0–0 through 6–6) is assigned a probability, forming a 7×7 grid. Every betting market is simply a sum of cells in that grid.
